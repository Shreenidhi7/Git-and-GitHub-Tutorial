**Download GIT**
Git Home Page - https://git-scm.com/
Windows Installer - https://git-scm.com/install/windows

During the Git Installation, enable the below
- Click on the Checkbox - Add a Git Bash Profile to Windows Terminal [ This option has been newly added by Git ]
	- Basically Git Bash is a kind of terminal which is reliable as compared to other terminals when using git.
		- Command prompt is not actually very good to perform git commands
			- Obviously all git commands works fine, but certain commands like **ls** and few other wont run.
	- Git Bash is the ideal terminal to run GIT Commands specially for Windows.
- Click on Next
- Choosing the default editor used by Git --> Select Visual Studio Code as Set default editor
	- By default it would suggest -> Use Vim
	- But change it to Visual Studio Code
- Next all the things are fine, don't make any changes and click on Next and complete the installation
- Click on Launch the Git Bash and explore the Git Bash Terminal UI
