
What is a Log?
1. Its a kind of details/information/metadata
2. Logs are something from which we get some more information
3. Assume we are performing some operations and logs are tracking information
	1. Its like logs are storing all the information what we are actually doing.
		1. If we want to look at all those information, we simply go to the log section

We wont be looking at .git folder for logs. But where do we find these logs?
Command ->  git log

But this is not recommended to use this command. There is a better command
Command ->git log --oneline

```
PS D:\Git and GitHub\Git_Practical_Tutorial> git log --oneline
1dd6575 (HEAD -> master) new iot data ingested
28669ec Data Ingest and IOT Data Ingestion
b10dd53 Ingest Data
08a8756 initial commit
PS D:\Git and GitHub\Git_Practical_Tutorial> 
```

Latest Commit is also known as "HEAD" of the current branch
