# Git and GitHub Tutorial

A beginner-friendly repository to learn Git and GitHub step by step, from fundamentals to advanced workflows.

## Topics Covered
- Git basics
- Repository initialization
- Commits
- Branching and merging
- Merge conflicts
- Rebasing
- Stashing
- Working with remotes
- GitHub pull requests
